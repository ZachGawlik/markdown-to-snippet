# My snippets

## Logs

### Log

Print statement

> prefix: `lg`

```js
console.log($1);
```

### Log variable

> prefix: `lgv`

Print variable name and value

```js
console.log(`$1: ${$1}`);
```

### Prefix alone in description

`lg`

```js
console.log($1);
```

### Prefix mixed in description

`lg`. Logs the variable

```js
console.log($1);
```

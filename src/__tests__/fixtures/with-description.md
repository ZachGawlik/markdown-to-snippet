### Log

Print statement

> prefix: `lg1`

```js
console.log($1);
```

### Log variable

> prefix: `lgv`

Print variable name and value

```js
console.log(`$1: ${$1}`);
```

### Prefix alone in description

`lg2`

```js
console.log($1);
```

### Prefix mixed in description

`lg3`. Logs the variable

```js
console.log($1);
```
